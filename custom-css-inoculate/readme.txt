== Description ==
Add Custom CSS styles to your site without modifying the theme files.
This approach will useful for Global CSS to site,This is perfect for adding custom css small Tweaks.
This approach will add custom CSS design to site immediately Globally. 

Key Features:

1. Add Custom CSS without modifying Theme Files.

2. Useful to add small CSS Tweaks.

3. Immediately apply CSS for Global Site Design Changes

== Installation ==
Follow the below instructions to install this plugin to your WordPress website.

Ex.

1. Upload to the `WP-content/plugins/` directory.
2. Activate the plugin through the \\\'Plugins\\\' menu in WordPress.
3. Go to the setting section and open Custom CSS inoculate.
4. Add your CSS Code and Save Changes.
5. Done!!..
